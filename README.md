# TechAuto-MRP

### Problema
  Atráves de algumas conversas, foi identificado uma oportunidade de melhoria no processo de planejamento de matéria-prima da linha de produção de uma empresa. Esse processo é crítico, porém é feito manualmente por um funcionário com vários arquivos Excel, onde demanda muitas horas de trabalho do mesmo.

### Motivação 
  Hoje, o processo de planejamento de matéria-prima é o processo responsável por garantir que a linha de produção sempre terá matéria-prima para produzir as demandas da fábrica, sendo assim, o processo se torna critíco. O processo de planejamento começa  com o time de compras que é responsável por fechar os contratos com os fornecedores baseados em dados historicos de vendas, após isso, o analista de planejamento faz uma compra de matéria-prima para 3 meses de produção. Ao longo do tempo, as demandas de vendas vão começar a chegar, assim o analista de planejmaento colete os pedidos de vendas e executa o calculo para obter a quantidade de matéria-prima necessaria para produzir a demanda do cliente. Esse processo é feito manualmente pelo analista de planejamento que usa várrias planilhas Excel para calcular a quantidade de matéria-prima para comprar.

### Proposta de Solução
  A proposta para esse problema é criação de um sistema que automatize esse processo de planejemanto, levando em conta vários requisitos e cenários. Assim, o processo será padronizado e uma quantidade enorme de horas de trabalho do funcionário será salva.
